# Services Page

## Page Goal

Convert a broad service list into clear, client-friendly offerings that help visitors understand fit and next steps.

## Required Sections

1. Page intro explaining the overall service approach.
2. Dedicated blocks for Interior Design, Space Planning, Furniture Selection, Color Consultation, and Project Management.
3. Segment cues for residential and commercial applicability.
4. Delivery model or engagement expectations section.
5. Contact CTA with invitation to discuss project scope.

## UX Notes

- Service cards or sections should make scanning easy on mobile.
- Avoid technical jargon that assumes design-industry familiarity.
- If FAQs are added later, they should appear below the service blocks rather than interrupting the primary explanation flow.
