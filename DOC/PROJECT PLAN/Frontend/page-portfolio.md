# Portfolio Page

## Page Goal

Use visual proof to build trust and help prospects imagine the quality and range of Three Circles work.

## Required Sections

1. Portfolio intro with positioning copy.
2. Filter or grouping controls for residential and commercial work if enough content exists.
3. Responsive project grid with strong imagery and concise project summaries.
4. Optional highlighted case-study strip for flagship work.
5. CTA to Contact Us.

## State Rules

- Loading state should preserve card layout rhythm.
- Empty state should explain that portfolio items are being curated if no projects are yet published.
- Failed image or data load should preserve page usability and allow retry.

## Assumption

- Assumption: portfolio cards link to `/portfolio/[slug]` detail pages when a project detail model is approved.
