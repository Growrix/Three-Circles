# About Us Page

## Page Goal

Explain who Three Circles is, how the team approaches interior design, and why clients should trust the studio.

## Required Sections

1. Intro hero with studio story and tone-setting imagery.
2. Mission and vision block using the confirmed source-plan language.
3. Core values section with practical interpretation of creativity, quality, client focus, integrity, and collaboration.
4. Process overview showing how projects typically move from discovery to delivery.
5. Optional founder or studio profile section if approved content exists.
6. CTA to Services or Contact Us.

## UX Notes

- The page should feel more narrative than promotional.
- Process content must reduce uncertainty for prospects who are new to hiring a designer.
