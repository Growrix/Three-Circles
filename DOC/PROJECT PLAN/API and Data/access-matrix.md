# Access Matrix

| Surface or API | Anonymous Public | Internal Editor | Internal Admin |
| --- | --- | --- | --- |
| View public services | Allow | Allow | Allow |
| View published portfolio index | Allow | Allow | Allow |
| View published portfolio detail | Allow | Allow | Allow |
| Submit contact inquiry | Allow | Allow | Allow |
| Create or edit service offering | Deny | Allow | Allow |
| Create or edit portfolio project | Deny | Allow | Allow |
| Upload or replace portfolio assets | Deny | Allow | Allow |
| View inquiry records | Deny | Allow | Allow |
| Change inquiry status | Deny | Allow | Allow |
| Manage users or roles | Deny | Deny | Allow |

## Notes

- The editor and admin roles are planning assumptions for internal operations.
- Anonymous access is restricted to public read models and contact submission only.
