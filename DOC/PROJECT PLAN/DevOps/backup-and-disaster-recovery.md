# Backup And Disaster Recovery

## Backup Scope

- Structured content data.
- Inquiry records.
- Media metadata.
- Uploaded media assets or their authoritative storage references.

## Recovery Rules

- Define database backup cadence before production launch.
- Verify restore procedures in staging or equivalent safe environment.
- Recovery plans must include both data restore and media-link integrity validation.

## Disaster Recovery Priorities

- Restore public site availability.
- Restore inquiry capture capability.
- Restore portfolio media integrity.
