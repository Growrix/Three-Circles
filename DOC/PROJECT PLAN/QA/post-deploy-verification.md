# Post Deploy Verification

## Immediate Checks

- Verify the five required public routes load successfully.
- Submit a production-safe contact test and confirm persistence plus notification.
- Confirm portfolio media loads without broken assets.
- Confirm analytics and alerting signals are visible.

## Follow-Up Checks

- Review logs for unexpected validation or authorization failures.
- Confirm no draft content is exposed publicly.
- Confirm rollback instructions are current if issues are found.
