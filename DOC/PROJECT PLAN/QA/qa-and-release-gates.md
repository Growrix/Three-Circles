# QA And Release Gates

## Release Gates

- No P0 or P1 defects open.
- All required public pages render correctly in production-like validation.
- Contact form success and failure paths are verified.
- Accessibility and core responsive checks pass.
- Monitoring and alerting are configured for production.

## Blockers

- Broken navigation or missing required page content.
- Contact submission data loss.
- Public exposure of internal data.
- Major media delivery failures on portfolio routes.
