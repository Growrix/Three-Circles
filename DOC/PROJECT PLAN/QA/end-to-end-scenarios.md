# End To End Scenarios

## Critical Scenarios

1. Visitor lands on Home, explores Services, views Portfolio, and submits Contact form successfully.
2. Visitor submits invalid Contact form data and receives recoverable inline validation.
3. Visitor experiences a temporary server failure on Contact submission and can retry without losing entered data.
4. Anonymous user attempts to access assumed admin routes and is blocked.
5. Newly published portfolio project appears on the public portfolio route and hidden drafts do not.

## Additional Checks

- Navigation consistency across all required pages.
- Mobile layout integrity for navigation, grid content, and form controls.
- Dark and light theme readability if both themes are implemented.
